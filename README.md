# 🧮 BMI Calculator Web App

A simple and interactive **Body Mass Index (BMI) Calculator** built using **React (Vite)**.  
This project allows users to calculate their BMI based on height and weight and understand their health category instantly.

---

## 🚀 Live Demo
👉 https://bmi-calculator-eosin-two.vercel.app

---

## 📌 Features

- ⚡ Fast and responsive UI
- 📊 Real-time BMI calculation
- 🧍 Input height and weight easily
- 📱 Mobile-friendly design
- 🎯 Instant health category result (Underweight / Normal / Overweight)

---

## 🛠️ Tech Stack

- React (Vite)
- JavaScript (ES6+)
- HTML5
- CSS3
